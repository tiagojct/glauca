# Specimen

Self-contained Glauca demos, no network needed. Open in a browser.

- `specimen.html` — type and colour specimen: IBM Plex Serif/Sans/Mono embedded,
  a dark/light toggle, the core tiers (frost field + the one blue mark), and a
  code panel showing the deliberate exception, where folium green joins the blue.
  Also carries a small Inter comparison for the sans.
- `type-specimen.html` — the type roles (display/headline/title/body/caption/code).
- `reading-comparison.html` — body-face reading comparison.
- `pt-specimen.html` — European Portuguese coverage.
- `motion.html` — the motion eases and durations (honours prefers-reduced-motion).
- `brand.svg`, `poster-proof.svg` — brand mark and poster proof (core hues only:
  the extended tier stays out of posters by design).
- `glauca-dataviz.png`, `glauca-dataviz-cvd.png` — data-viz scales and their
  colour-vision check.

All fonts are IBM Plex (OFL); the palette is Glauca (`glauca.json`).
